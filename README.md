# web-vize-gorkemhicyilmaz
Projenin amacı, kullanıcıların yapay zekâ uygulamalarını doğru ve etkili bir şekilde kullanabilmesini ve kullanıcıların yanlış prompt girmesini engelleyecek, yapay zekâ çeşitlerini ve kullanım yöntemlerini açıklayan, aynı zamanda hazır prompt örnekleri sunan bir web sayfası geliştirmek hedeflenmektedir. Görkem Hiçyılmaz tarafından tasarlanmıştır.
